Definiciones

Hecho
Es un antecedente específico con el cual se hace un razonamiento.

Hechos de inicio
Información con la que el usuario decide comenzar el razonamiento.

Objetivo
Hecho a ser demostrado.
Inferencia
Consecuente que ha sido probado mediante razonamiento.
Deducción (Encadenamiento hacia adelante)
Razonamiento que parte de los hechos iniciales dados para llegar a conclusiones usando la base de conocimientos.
Inducción(Encadenamiento hacia atrás)
Razonamiento que parte de un objetivo y busca probarlo usando los hechos iniciales, hechos inferidos y hechos preguntados.
Abducción
Razonamiento que hace deducción, intuición e inducción con las reglas intuidas hasta encontrar una inferencia positiva.
Intuición
Razonamiento por el cual se obtienen reglas que se cree podrían dispararse.
Suma probabilística
Proceso que se usa para conocer la probabilidad de que una regla sea inferida.
Umbral de disparo
Determina si la regla puede ser inferida o no.
Umbral de intuición
Determina si la regla puede ser intuida o no.
Álgebra de signos
Modelo utilizado para determinar el signo del resultado de la inferencia basado en un criterio.
Incertidumbre
Grado de confiabilidad de que un hecho es verdadero.
Base de conocimiento
Consiste en las reglas y datos intrínsecos al área del problema. Generalmente se construye a través de consulta con uno o varios expertos del área, para incluir el conocimiento heurístico, manuales, base de datos, etc.
Explicación
Desplegado de justificación a cerca de las operaciones, estructura y resultados de los procesos del sistema experto.
Abrir
Carga en memoria una base de conocimientos anteriormente elegida por el usuario.
Borrar
Elimina todos los datos en memoria.
